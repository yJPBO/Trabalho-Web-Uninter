Seja bem-vindo!, meu nome é João Pedro, tenho 21 anos, sou estudante do curso de Análise e Desenvolvimento de Sistemas da UNINTER e moro no interior de Pernambuco.

Como programador, meu foco está nas linguagens Java e Python, com experiência em programação orientada a objetos, estrutura de dados e desenvolvimento de software. Além disso, estou buscando aprimorar minhas habilidades em desenvolvimento web e bancos de dados, áreas que venho estudando para expandir meu conhecimento.

Sou apaixonado por tecnologia e inovação, e adoro escrever código para desenvolver minhas ideias. No meu tempo livre gosto de explorar novas tecnologias, desenvolver pequenos projetos para praticar e jogar alguns jogos cooperativos com meus amigos.

Atualmente, meu objetivo é aprimorar minhas habilidades como programador para ingressar no mercado de TI e contribuir com soluções inovadoras. Se quiser trocar uma ideia sobre tecnologia ou desenvolvimento, estou sempre aberto para conversar, basta acessar a pagina de contato!
